About Event Manager

Event Manager is a simple Flask application that demonstrates how to build dynamic web pages using Flask, Jinja templates, and Bootstrap.

Features

  • Create events
  • Update event information
  • Delete events
  • Responsive Bootstrap interface
This application is built using Flask, Bootstrap, and Jinja template inheritance.
